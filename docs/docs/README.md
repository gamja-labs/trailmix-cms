Welcome to Trailmix CMS documentation. Trailmix CMS is a powerful, type-safe content management system built for NestJS applications.

Start with **[`@trailmix-cms/core`](/core.html)** — the recommended, provider-agnostic foundation — then
add the auth provider ([better-auth](/better-auth.html) or [Clerk](/clerk.html)) and [email](/email.html)
you need.

Get started by reading the [Introduction](/introduction.html) or jump straight to [Installation](/installation.html).
